# Tool Defaults

Default tool bindings for all projects. Override in `.ai/tool-overrides.md` (only divergences).

| Category | Default Tool | Notes |
|----------|-------------|-------|
| browser | agent-browser | iOS projects: override with maestro |
| symbol_nav | serena | Semantic symbol search + memory |
| search | grep / glob | For directed searches; agent for open-ended |
| design | figma-mcp | Gate: UI Audit phase only |
| ticket | — | Set in project overrides |
| docs | context7 | Library documentation |
